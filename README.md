# Salsa

[![Build Status](https://travis-ci.com/RelationalAI-oss/Salsa.jl.svg?branch=master)](https://travis-ci.com/RelationalAI-oss/Salsa.jl)

[salsa-rs/salsa](https://github.com/salsa-rs/salsa).
